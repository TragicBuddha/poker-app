
# Poker Bankroll Tracker

React Native + Expo mobile app for tracking poker sessions, bankroll, and real-time analytics.

![Poker App Screenshot](assets/images/MainScreen.png)

## Features
- Track cash games and tournaments with session details (blinds, location, buy-in/out)
- Real-time bankroll tracking with hourly earnings and profit/loss calculations
- Custom charts visualizing cash vs tournament earnings
- Date/time pickers, location selectors, and responsive modal forms
- Firebase Firestore integration for live data sync
